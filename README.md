# PULSE – Mood Board App

## Objective
The objective of the PULSE Mood Board App is to design a modern and motivating fitness application UI using Figma. The app helps users stay active through workout inspiration, health tracking, stretching exercises, and healthy nutrition guidance.

## Project Description
PULSE is a fitness mood board designed to inspire users to maintain a healthy lifestyle. The design combines workout images, wearable fitness tracking, nutrition, and clean mobile UI references to create an attractive and user-friendly interface.

## Features
- Group Workout Inspiration
- Strength Training
- Fitness App UI Reference
- Wearable Heart Rate Tracking
- Stretching Exercises
- Healthy Nutrition Guidance

## Color Palette

### Primary Color – #FF5A1F
- Used for energy, motivation, and action.
- Represents strength and fitness.

### Secondary Color – #1F3873
- Used to create trust and stability.
- Gives the app a professional appearance.

### Accent Color – #B4FF3C
- Highlights important buttons and key elements.
- Represents health, freshness, and progress.

### Neutral Black – #1A1A1A
- Used for headings and text.
- Improves readability and contrast.

## Design Elements
- Fitness workout images
- Mobile app UI inspiration
- Smart wearable fitness tracker
- Healthy nutrition section
- Stretching exercise section
- Clean and modern card layout

## Tools Used
- Figma
